1️⃣ What is the difference between var, let, and const?

Ans : var is function scoped, let and const is blocked scoped, const can't be reassigned but value (object/array) can be updated and let can be reassigned

2️⃣ What is the spread operator (...)?

Ans : Spread operator হলো একটি ফিচার যেটা দিয়ে arrays, strings, অথবা  objects একটা indivisual এলিমেন্ট এ কনভার্ট করে

3️⃣ What is the difference between map(), filter(), and forEach()?

Ans : map() return's an new array with the result of original array, 
filter() return's an new array with elements that pass the condition,
forEach() iterate over an array
